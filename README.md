# 🛠️ DIGITOOL 

> Your one-stop marketplace for professional digital tools and premium subscriptions.

DIGITOOL is a streamlined e-commerce platform designed for creators and professionals. Whether you need a one-time asset or a recurring premium subscription, DIGITOOL provides a seamless checkout experience with a modern, high-performance interface.

---

## 🚀 Features

* **Efficient Purchasing** – A friction-less "Buy Now" flow that allows users to select and manage digital products with ease.
* **Tiered Access** – Support for multiple pricing models, including **Free**, **Monthly**, and **Yearly** premium subscriptions to fit every budget.
* **Fully Responsive** – Optimized for mobile, tablet, and desktop using a mobile-first design approach.

---

## 🛠️ Tech Stack

The project is built with a modern frontend stack focused on speed, utility-first styling, and smooth user feedback.

| Technology | Purpose |
| :--- | :--- |
| **React** | Component-based UI architecture |
| **Tailwind CSS** | Custom utility-first styling |
| **DaisyUI** | Beautifully themed accessible components |
| **React-Toastify** | Real-time, interactive notifications |
| **JavaScript (ES6+)** | Dynamic logic and state management |
| **JSON** | Localized data storage for product catalogs |

## 📦 Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/digitool.git](https://github.com/your-username/digitool.git)
    ```
2.  **Navigate to the directory:**
    ```bash
    cd digitool
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm run dev
    ```
